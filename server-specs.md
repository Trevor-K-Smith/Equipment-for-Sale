# Server Spec Sheet

## Chassis
- **GIGABYTE G292-Z45-00** — 2U Rack Mount
- 2x redundant PSU

## CPUs (x2)
- **AMD EPYC Rome Engineering Sample** (equivalent to EPYC 7742)
- 64 cores per socket, 128 cores total
- Zen 2, SP3 socket

## RAM (2 TB)
- 16x **Samsung M393AAG40M32-CAE** 128 GB DDR4 RDIMM ECC @ 3200 MT/s

## GPUs (x7)
- 2x **NVIDIA L4** — 24 GB GDDR6 each
- 5x **NVIDIA RTX A5000** — 24 GB GDDR6 each

## Storage
- **Samsung PM1733a** — 12.8 TB NVMe U.2 SSD
- **Samsung 870 QVO** — 8 TB SATA SSD

## Networking
- **Mellanox ConnectX-7** — up to 400 GbE
- 2x **Intel I350** — 1 GbE (onboard)
